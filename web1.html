Web Development 3.o

Web Development has gone through three phases so called as web1.o , web2.o and web3.o-the upcoming one.

web 1.o is the part where users can only read information.No interaction between the users and the site was there.

web 2.o , here the uses can read as well as write information and data from and to the website it mainly consists of -

  1. Front-End
  2. Back-End
  3. Database

web 3.o , the recent one focuses on shifting the ownership of the data of each and every users to everyone on the blockchain.

Note-The most favoured are to web2.o and web3.o , because each web development has its own adds and subtracts.
